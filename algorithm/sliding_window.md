# Sliding Window (슬라이딩 윈도우)

### 알고리즘
* 창문을 한쪽으로 밀면서 문제를 푸는 것과 유사해서 붙여진 이름
* 투 포인터처럼 구간을 훑으면서 지나가는 것은 유사하나 슬라이딩 윈도우는 부분 배열의 길이(구간)의 넓이가 고정적이다.
